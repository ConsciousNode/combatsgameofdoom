Combat's Game of Doom — CSS Engine

This is a fighting game where the game engine is strictly CSS. Not assisted by CSS. Not styled with CSS. CSS IS the engine.

Architecture: Independent fighter layers. Each fighter traverses their own parallel 100vw layer. Visual overlap occurs naturally without DOM collision.

CONTROLS

PLAYER 1

  • A / D : Move Left / Right
  • W : Jump
  • S : Block (Negates damage)
  • F : Light Attack
  • G : Heavy Attack

PLAYER 2

  • Arrows : Move & Jump & Block
  • L : Light Attack
  • ; : Heavy Attack

FIGHTERS

PAZUZU: Glass cannon. Attacks do high damage to Oni, but takes severe damage from heavy hits.

ONI: Slow tank. Natural armor means it takes two light attacks from Pazuzu to register 1 tier of damage.
